Демонстрація використання селекторів HTML і CSS

Селектори тегу

HTML-код:

<section> <h2>Привітання</h2> <p>Ласкаво просимо до нашого магазину!</p> </section>

CSS-код:

section { border: 2px solid #ffcc00; padding: 10px; background-color: #fff3cd; }

Результат:

Привітання

Ласкаво просимо до нашого магазину!

Селектори класу

HTML-код:

<div class="product"> <h3 class="title">Корм для котів</h3> <p class="description">Високоякісний корм.</p> </div>

CSS-код:

.product { border: 1px solid #ddd; padding: 10px; } .title { color: #ff6600; font-size: 1.2rem; } .description { color: #555; }

Результат:

Корм для котів

Високоякісний корм.

Селектори ідентифікатора

HTML-код:

<div id="promo"> <p>Сьогодні знижка 10%!</p> </div>

CSS-код:

#promo { background-color: #ffecb3; padding: 10px; border: 1px dashed #ff9800; }

Результат:

Сьогодні знижка 10%!

Інші селектори

HTML-код:

<ul> <li>Коти</li> <li>Собаки</li> </ul>

CSS-код:

li + li { color: #ff6600; } ul > li { font-weight: bold; }

Результат:

  • Коти
  • Собаки
